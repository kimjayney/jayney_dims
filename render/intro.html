End-to-End Encrypted Location Service

Location data security features:

  • Server stores only encrypted location data
  • Client-side decryption
  • Unique IV for each location point
  • Private key required for viewing

Our promise:

  • We will not collect location data by decrypting it.
  • We will also not collect IP addresses.
  • We will deploy on a stable infrastructure and manage it so that location information can be stored securely.
